## Key Features

* Login 
  - Municipal Agriculture Office
  - Farmer Portal

* Municipal Agriculture Office 
  - Dashboard
  - Insurance Application
  - Damage Assessment
  - Farmers
  - Logout

* Farmer Portal
  - Dashboard
  - Insurance Status
  - My Profile
  - Documents
  - Transaction History
  - Contact LGU
  - Help
  - Logout
